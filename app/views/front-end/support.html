Overview of tabb.io

This is how yoour customer will experience tabb.io:

  • The customer will browse your store and choose a product they are interested in.
  • They will then proceed to checkout and choose "tabb.io" as a payment method.
  • Once your customer clicks the checkout button, you will query the tabb.io API with the order details and customer information.
  • Tabb.io will respond with an acceptance or refusal based on the information sent over.
  • The customer now has 14 days to pay without any charges!

Using the API

There are a few steps you must take in order to use the tabb.io API:

  • Sign up for a merchant account.
  • Request your staging and production environment IP's get whitelisted.
  • Use the API key that tabb.io gives you - you will receive different API keys for the different environments.

Verification Checklist

Before moving on to integration, please make sure you can access both our query and command methods from both staging and production:

  • Staging
  • http://staging.merchantapi.tabb.io/command/api
  • http://staging.merchantapi.tabb.io/query/api
  • Production
  • http://merchantapi.tabb.io/command/api
  • http://merchantapi.tabb.io/query/api
