Create a Loan Request

The loan request methods will allow you to create a consumer loan with tabb.io. When you create this loan request, tabb.io will perform fraud and risk decisions on the consumer and then let you know whether the purchase is approved or not.

A

Create Object

Configure the library to communicate with tabb.io.
B

Add Consumer

Tell tabb.io who is making the purchase. Required.
C

Add Cart Items

Let tabb.io know whats in the cart to improve approval rates.
D

Make Request

Make the API request once everything above is filled in!
